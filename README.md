# KitchenKing-iOS

一个基于 SwiftUI 的 iOS 菜谱生成应用，让用户输入食材后，由不同菜系的名厨为其创造独特菜谱。

## 🌟 功能特性

### 核心功能
- **食材输入**：用户可以输入拥有的食材
- **智能生成**：AI 根据食材生成多种菜系的菜谱
- **厨师展示**：不同菜系厨师同时创作，展示制作过程
- **菜谱详情**：查看完整的制作步骤、食材搭配和小贴士

### 用户体验
- **像素风格设计**：复古游戏风格的 UI 界面
- **实时状态更新**：显示厨师制作进度和状态
- **响应式交互**：流畅的动画和用户反馈
- **随机食材**：一键生成随机食材组合

## 🎨 设计特色

### 视觉风格
- **像素艺术**：采用复古游戏像素风格设计
- **SF Symbols**：使用苹果官方图标库
- **SwiftUI**：现代化声明式 UI 框架
- **动画效果**：流畅的状态转换和加载动画

### 界面组件
- **HeaderView**：应用标题和皇冠图标
- **IngredientInputView**：食材输入区域
- **ChefGridView**：厨师网格展示
- **ChefCardView**：单个厨师卡片
- **DishDetailView**：菜谱详情展示

## 🚀 快速开始

### 环境要求
- macOS 14.0+
- Xcode 15.0+
- iOS 17.0+

### 安装步骤
1. 克隆项目
   ```bash
   git clone https://github.com/AissenLiu/KitchenKing-iOS.git
   ```

2. 打开项目
   ```bash
   cd KitchenKing-iOS
   open KitchenKing-V2.xcodeproj
   ```

3. 构建运行
   - 选择目标设备或模拟器
   - 点击运行按钮 (⌘+R)

## 🎯 使用方法

### 基本操作
1. **输入食材**：在输入框中输入拥有的食材，用逗号分隔
2. **开始创作**：点击"开始厨王争霸"按钮
3. **查看进度**：观察不同菜系厨师的创作过程
4. **选择菜谱**：点击完成的菜品查看详细制作步骤

### 高级功能
- **随机食材**：点击"随机食材"获取灵感
- **重新开始**：点击重置按钮清空当前状态
- **菜品详情**：查看完整的制作步骤和营养信息

## 🎮 游戏化元素

### 厨师状态
- **空闲中**：等待食材输入
- **制作中**：正在创作菜谱
- **已完成**：成功创作完成
- **失败**：创作遇到困难

### 完成顺序
- 显示不同菜系厨师的完成时间
- 统计制作状态和成功率
